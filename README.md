# mcp_repos

A collection of [MCP](https://modelcontextprotocol.io) servers I build while
learning the Model Context Protocol. Each server lives in its own subdirectory
and is published independently to PyPI.

## Servers

| Server | Directory | PyPI | What it does |
|--------|-----------|------|--------------|
| **todo-mcp-server** | [`todo-mcp-server/`](todo-mcp-server/) | [`todo-mcp-server-kit`](https://pypi.org/project/todo-mcp-server-kit/) | A plaintext todo list exposed over MCP (CRUD + tags + search + stats) with a browser dashboard |

_More to come._
